Exception - breaks execution of code, it can be a bug or issue created by Syntax Error, Runtime Error or Logical Error. try...catch - used to handle the error-generating part of code or applied by proffesinals where code is written complex. try - block used to test the code and if getting some error, it passes to catch block. catch - block used to handle failed code and provides an alternative solution of code. ex. redirect to another route. Syntax: try{ expression; } // code to be written or test catch(error){ expression; } // alternative solution of code Exception Handling - try...catch
